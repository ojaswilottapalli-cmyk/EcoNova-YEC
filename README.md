🌱 ECONOVA – Trash 2 Trend
♻️ Turning waste into aesthetic products while helping the planet
🌍 About the Project
Econova is a sustainability initiative that transforms everyday waste materials into stylish and useful products like bookmarks and keychains.
Our goal is to promote eco-friendly habits in a creative and fun way.
✨ Features
🚀 Smooth scrolling navigation
🎨 Aesthetic eco-friendly design
🖱️ Hover animations & effects
📱 Responsive layout (works on devices)
🌿 Clean and minimal UI
🛠️ Built With
HTML5
CSS3
JavaScript
📂 Project Structure
econova/
│── index.html   # Main webpage
│── styles.css   # Styling and design
│── script.js    # Interactivity
▶️ How to Run

Open index.html in your browser
